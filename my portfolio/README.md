## üîß Customization

### Adding New Sections
1. Create new section div in `index.html`
2. Add corresponding CSS styles in `styles.css`
3. Update navigation links
4. Add JavaScript functionality in `script.js` if needed

### Adding New Skills
1. Add new skill card in the skills section
2. Update skill bar width in CSS
3. Style with appropriate icons and descriptions

### Adding New Projects
1. Add new project card in the projects section
2. Configure GitHub and live demo links
3. Add appropriate Mario-themed icons
